## How to Run

### 1. Backend
```bash
cd backend
npm install
copy .env.example .env
npm run dev
```

Edit `.env` with your MySQL password before running.

### 2. MySQL
Open MySQL Workbench and run:

```sql
CREATE DATABASE digital_guide_complete;
```

Then import/run `backend/database.sql`.

### 3. Frontend
Open another terminal:

```bash
cd frontend
npm install
npm run dev
```

Open:
http://localhost:5173
